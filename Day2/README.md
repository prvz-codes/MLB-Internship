# Day-2: Student Record Management System

## Overview

This project is a console-based Student Record Management System developed using Python. It stores student records in memory using a list of dictionaries and allows users to manage student information through a menu-driven interface.

## Features

* Add a new student.
* View all student records.
* Search for a student using their Roll Number.
* Update an existing student's information.
* Delete a student record.
* Display the total number of students.
* Prevent duplicate Roll Numbers from being added.

## Concepts Used

* Functions
* Lists
* Dictionaries
* Loops
* Conditional Statements
* User Input
* Menu-Driven Programming

## What I Learned

* How to use lists and dictionaries together to manage data.
* How to organize a program using functions.
* How to perform CRUD (Create, Read, Update, Delete) operations.
* How to search, update, and delete records using loops and dictionaries.
* The importance of validating user input, such as checking for duplicate Roll Numbers.

## Challenges Faced

* Managing student records using appropriate data structures.
* Implementing update and delete operations correctly.
* Preventing duplicate student entries based on Roll Number.
* Designing a simple and interactive menu-driven application.

## Conclusion

This project helped me strengthen my understanding of Python data structures, functions, and control flow by building a practical console-based application.
