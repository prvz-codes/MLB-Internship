# 📊 Student Records Management using Pandas

## 📌 Project Overview

This is a simple Python console application that uses the **Pandas** library to analyze student records stored in a CSV file. The program provides a menu-driven interface to display and filter student information based on different criteria.

---

## 🚀 Features

* Display all student records
* View the topper student
* Display the top 10 students
* Display the bottom 10 students
* Show students with the best attendance
* Show students with the highest marks in Chemistry
* Show students with the highest marks in Physics
* Show students with the highest marks in Computer
* Interactive menu for easy navigation

---

## 🛠️ Concepts Used

* Reading CSV files using `pd.read_csv()`
* DataFrames
* Sorting data using `sort_values()`
* Filtering data with conditions
* Selecting specific columns
* Using `head()` and `tail()`
* Functions
* Loops (`while`)
* Conditional statements (`if-elif`)

---

1. Install Pandas:


## 🎯 Purpose

This project was created to practice basic **Pandas DataFrame operations**, including sorting, filtering, and displaying data from a CSV file through a menu-driven Python application.
