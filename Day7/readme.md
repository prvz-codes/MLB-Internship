# Day 7 - Iris Flower Classification

## What is Classification?

Classification is a machine learning technique used to predict a category or class. In this project, the model predicts the species of an Iris flower.

## Difference between Regression and Classification

* **Regression** predicts numbers (for example, student average marks).
* **Classification** predicts categories (for example, Setosa, Versicolor, or Virginica).

## Evaluation Metrics Used

* **Accuracy** – Checks how many predictions are correct.
* **Precision** – Checks how many predicted classes are actually correct.
* **Recall** – Checks how many actual classes were correctly identified.
* **F1-Score** – Gives a balanced score using Precision and Recall.
* **Confusion Matrix** – Shows correct and incorrect predictions for each class.

## Model Performance and Observations

The Logistic Regression model performed very well on the Iris dataset. It achieved high accuracy, precision, recall, and F1-score. The confusion matrix showed that almost all flowers were classified correctly.
