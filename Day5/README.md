# Student Performance Dashboard

## Overview
This project is a simple Student Performance Dashboard built using **Python**, **Pandas**, **Matplotlib**, and **Seaborn**. It analyzes a student dataset and visualizes the results using different charts.

## Features
- Display total number of students.
- Calculate average marks of each subject.
- Show the top 5 performing students.
- Show students who need improvement.
- Find the subject with the highest class average.
- Visualize data using charts.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations
- Bar Chart
- Histogram
- Scatter Plot
- Pie Chart
- Box Plot

## How to Run
1. Install the required libraries:
   ```bash
   pip install pandas matplotlib seaborn
   ```

2. Place `cleaned_student_data.csv` in the project folder.

3. Run the program:
   ```bash
   python studentPerformanceDashboard.py
   ```
